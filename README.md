# Revenue Agent Team Starter Kit

The exact 4-agent Claude Code system that turned "giant teddy bears" into a positioned offer, a YouTube angle, and a follow-up sequence — in one live run, no editing.

## What's inside

- `lead-magnet/` — the four agent files (`market-signal-researcher`, `offer-architect`, `content-angle-strategist`, `conversion-system-builder`), setup guide, annotated demo run, and a worksheet for swapping in your own business idea.
- `runbooks/` — operating notes for running the revenue agent pipeline.
- `outputs/` — sample outputs from live runs of the agent team.
- `business-brief.md` — the business context the agents were built around.

## Quick start

1. Read `lead-magnet/setup-guide.md` and copy the four agent files into your project.
2. Restart Claude Code so it picks up the new agents.
3. Open `lead-magnet/swap-in-worksheet.md` and swap in your own business idea.
4. Run the four agents in order: research → offer → content → conversion.

See `lead-magnet/README.md` for the full breakdown.
